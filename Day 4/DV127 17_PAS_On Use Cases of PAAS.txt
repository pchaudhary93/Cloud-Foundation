17_SAS_On Use Cases of PAAS.




	Q1
	What is PAAS Platform as a service?
	ANS
	It is a hosting model in which instead of providing the basic operating system users will be provided with the platform on which the user needs to build something for the end users. 


	Q2
	What is PAAS referred to as _______________?
	ANS
	Serverless design
